# Sistema de Gestão de Obras - Personas

Este projeto apresenta uma interface web simples desenvolvida para representar visualmente duas personas do sistema de gestão de obras. A proposta foi criada com foco na organização das informações dos perfis de usuários, destacando características, objetivos, necessidades, dores, mapa de empatia e justificativa de adesão ao sistema.

## Objetivo

O objetivo deste projeto é demonstrar, de forma visual e organizada, os principais perfis de usuários envolvidos no contexto de um sistema de gestão de obras, contribuindo para a etapa de levantamento de requisitos e design de experiência do usuário.

## Estrutura do Projeto

O projeto é composto por:

- **Página inicial (`index.html`)**
  - Apresenta uma visão geral do projeto
  - Permite navegar para as páginas de cada persona

- **Página da Persona 1 (`persona1.html`)**
  - Representa **João Silva**, Engenheiro Civil
  - Perfil voltado ao planejamento, controle e acompanhamento da obra

- **Página da Persona 2 (`persona2.html`)**
  - Representa **Carlos Mendes**, Mestre de Obras
  - Perfil voltado à execução e supervisão das atividades no canteiro de obras

- **Arquivos de estilo (`style.css` e `reset.css`)**
  - Responsáveis pela estrutura visual, organização dos blocos e responsividade da interface

## Personas Apresentadas

### João Silva - Engenheiro Civil
Usuário com perfil mais analítico e estratégico, responsável pelo planejamento, monitoramento e controle das atividades da obra.

### Carlos Mendes - Mestre de Obras
Usuário com perfil mais prático e operacional, responsável pela execução, supervisão e atualização das tarefas realizadas no canteiro de obras.

## Funcionalidades

- Navegação entre páginas
- Exibição organizada das personas em blocos visuais
- Estrutura responsiva para dispositivos móveis
- Apresentação de:
  - dados demográficos
  - perfil comportamental
  - objetivos
  - necessidades
  - dores
  - mapa de empatia
  - justificativa de adesão

## Tecnologias Utilizadas

- **HTML5**
- **CSS3**

## Responsividade

O projeto possui ajustes para telas menores por meio de **media queries**, adaptando:

- menu de navegação
- layout em grid
- botões da página inicial
- largura do container
